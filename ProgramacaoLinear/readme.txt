Programação linear é um método de cálculo com sistemas de equações e inequações lineares que retorna a solução ótima (máxima ou mínima), caso o sistema a possua.

ProgramacaoLinear é o JFrame principal que serve de interface para o usuário inserir os dados da programação linear a duas variáveis.

Resolve e devolve as soluções ótimas, gera gráficos e testa pares ordenados.

O teste permite saber se um par ordenado qualquer passa em todas as restrições e, caso sim, qual o resultado.

Função objetivo — Modelo: Ax + By + K = Objetivo.
Restrições — Modelo: ax + by = c.

Operadores relacionais:

Use "=", ">=", "<=".

Para ">" e "<", use artifícios com ">=" e "<=".

Caso gerando gráficos ou testando pares ordenados, pode-se usar ">", "<" ou ("<>" ou "!=").
